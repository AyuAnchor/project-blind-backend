Flask YOLOv8 Object Detection API

Welcome

Welcome to the YOLOv8 Object Detection API. Below is how you can use the API:

Endpoints

/detect

Method: POST

Parameters:

  • image (file): The image file to be processed. Required
  • download (optional, boolean): If true, the API returns the image with detections. Default is false.

Example requests:

  • To receive JSON data:
    • curl -X POST https://projectblind.pythonanywhere.com/detect -H "Content-Type: multipart/form-data" -F "image=@images/cycles.jpg"
  • To download the image with detections:
    • curl -X POST https://projectblind.pythonanywhere.com/detect?download=true -H "Content-Type: multipart/form-data" -F "image=@./images/cycles.jpg" --output detected_image.jpg

Project Structure

  • app.py: Main Flask application file.
  • helpers.py: Contains image processing and drawing functions.
  • yolo_utils.py: Contains YOLO model and detection functions.
  • requirements.txt: List of Python dependencies.
  • .gitignore: Specifies files and directories to ignore in Git.
  • README.md: Project documentation file.
